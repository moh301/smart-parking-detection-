Smart Parking Detection System

Overview
This project is an AI-based Smart Parking Detection system that uses Computer Vision to detect available parking spots from images or video streams.

Features
- Detects occupied and empty parking spots
- Works on real-world parking lot images
- Uses Computer Vision techniques
- Can be extended for real-time applications

Technologies Used
- Python
- OpenCV
- NumPy
- Machine Learning / Computer Vision

How to Run
1. Clone the repository:
git clone https://github.com/yourusername/smart-parking-detection.git

2. Install dependencies:
pip install -r requirements.txt

3. Run the project:
python main.py

Future Improvements
- Real-time detection using camera
- Web dashboard
- Integration with IoT systems
